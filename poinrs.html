OBJECTIVES OF CLOUD COMPUTING:



1. Cost Reduction : Minimize capital expenses and optimize operational costs.
2. Flexibility and Scalability : Provide a flexible and scalable infrastructure.
3. Global Accessibility : Enable users to access resources globally.
4. Innovation and Agility : Facilitate rapid deployment and foster innovation.
5. Collaboration and Efficiency : Enhance collaboration and streamline business processes.



ADVANTAGES OF CLOUD COMPUTING:



1. Scalability : Easily scale resources based on demand.
2. Cost-Efficiency : Pay-as-you-go model reduces upfront.
3. Accessibility : Resources accessible from anywhere with an internet connection.
4. Resource Utilization : Efficient sharing of resources among multiple users.
5. Reliability : High levels of reliability and uptime from reputable providers.
6. Security : Robust security measures implemented by cloud providers.



DISADVANTAGE OF CLOUD COMPUTING:



1. Dependency on Internet :.Relies heavily on internet connectivity.
2. Limited Customization : Some services may limit customization options.
3. Data Security Concerns : Persistent concerns about data security and privacy.
4. Costs Over Time : Long-term costs can accumulate.
5. Limited Control : Users have limited control over the underlying infrastructure.



USES AND OTHER THINGS EXPLAINED:



Cloud computing is like renting a computer or a service over the internet instead of owning and managing your own physical computers.Cloud computing is used as big data analytics which helps processing large datasets efficiently or utilizing specialized techniques and technologies to analyze and extract meaningful insights from massive volumes of data. It emphasizes dealing with data at a scale that goes beyond the capabilities of conventional databases and analytical tools. Large datasets can range from terabytes to petabytes of information.It is also called as Internet of things(IoI) which means the network of interconnected physical devices, vehicles, appliances, and other objects embedded with sensors, software, and network connectivity. These devices can collect and exchange data, forming a connected ecosystem.
